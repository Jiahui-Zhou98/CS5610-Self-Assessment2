# Boston Airbnb Listings

A dynamic web application that displays Airbnb listings from Boston, loaded from a JSON file using modern JavaScript (fetch API and async/await).

## Project Objective

This project demonstrates a simple, beginner-friendly web application that:
- Fetches listing data from a JSON file using AJAX (asynchronous JavaScript)
- Displays 50 Airbnb listings in a responsive grid layout
- Shows complete listing information including images, prices, ratings, amenities, and host details
- Provides a clean, modern UI using Bootstrap 5 for responsive design
- Uses pure DOM API for creating and manipulating HTML elements

## Tech Requirements

- **HTML5**: Structure and semantic markup
- **CSS3**: Custom styling and responsive design
- **JavaScript (ES6+)**: 
  - Fetch API for AJAX requests
  - Async/await for asynchronous operations
  - DOM API for element creation and manipulation
- **Bootstrap 5**: Responsive grid system and UI components
- **Modern Web Browser**: Chrome, Firefox, Safari, or Edge (latest versions)

### Dependencies

- Bootstrap 5.3.8 (via CDN)
- No build tools or package managers required for basic functionality
- ESLint and Prettier (optional, for code quality)

## How to Install/Use

### Installation

1. **Clone or download the repository**
   ```bash
   git clone <repository-url>
   cd airbnb
   ```

2. **No additional installation required!**
   - The project uses Bootstrap via CDN
   - All dependencies are included or loaded from CDN
   - The `listings.json` file is already included

### Usage

1. **Open the application**:
   - **Option 1**: Simply open `index.html` in a web browser
   - **Option 2**: Use a local server (recommended for development):
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js http-server
     npx http-server
     
     # Using PHP
     php -S localhost:8000
     ```
     Then navigate to `http://localhost:8000`

2. **View Listings**: 
   - The page automatically loads and displays 50 listings from `listings.json`
   - Browse listings in a responsive grid (3 columns on desktop, 2 on tablet, 1 on mobile)
   - Each card shows: image, price, title, host info, rating, description, amenities, and property details

3. **Create Listing Form** (UI only):
   - The form on the right side allows you to enter listing details
   - Currently displays UI only (functionality not implemented)

## Code Overview

### Main Functions

- `loadListings()`: Fetches data from `listings.json` using async/await
- `displayListings()`: Renders all listings to the page
- `createListingCard()`: Creates individual listing card elements using DOM API
- `truncateText()`: Helper function to truncate long descriptions
- `getStarRating()`: Helper function to convert numeric rating to star display

### Key Technologies Used

- **AJAX**: `fetch()` API with `async/await` syntax
- **DOM Manipulation**: Pure JavaScript DOM API (no innerHTML for security)
- **Responsive Design**: Bootstrap 5 grid system (`col-12`, `col-md-6`, `col-lg-4`)
- **Error Handling**: Image error events and fetch error catching
